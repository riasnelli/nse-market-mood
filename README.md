# NSE Market Mood PWA

A Progressive Web App that shows real-time NSE market sentiment.

## Features
- Real-time NSE data
- Market mood indicator
- PWA (installable)
- Auto-refresh every 30 seconds
- Free hosting

## Deployment

### Netlify (Recommended)
1. Fork this repository
2. Go to [netlify.com](https://netlify.com)
3. Connect your GitHub repository
4. Deploy!

### Manual ZIP Upload
1. Download this project as ZIP
2. Go to [netlify.com](https://netlify.com)
3. Drag & drop the ZIP file
4. Deploy!

## Local Development
Just open `index.html` in a browser

## Data Source
- NSE India Official API
- Real-time NIFTY 50 data
- Market breadth indicators
